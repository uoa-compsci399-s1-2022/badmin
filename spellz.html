About

Spellz is an engaging proofreading game, aimed at helping people of all ages with their spelling!

Select Game

You can choose to play the daily challenge or pick a specific category and genre!

The daily challenge is ready to go, simply just click the text! Otherwise, feel free to pick a genre and category!

Fuzzy Highlight

While in play, the user will notice that while typing the text is highlighted and show changes on screen real-time.

The fuzzy highlight is only for words which have a certain degree of similarity between itself and word in input box.

The word on blue highlight is where entered input will change. The user must use arrow keys to get the blue highlight to the intended word which should be changed.

Game Mechanics

Score:

The score accumulates on the basis of combo, hint used and time. In the case the user enters wrong correction in the input box the score deducts by 30 points.

Not taking combos into account, by default one correct input increases score by 100 points

Combos:

To add to the combo the user needs to enter correct inputs within 5 seconds from each other.

By default each user is on combo 1 and gets 100 points for each correct input.

When reaching a combo 2 from combo 1, by entering next correct input within 5 seconds, a user gets an additional 200 points instead of standard 100 points.

The same concept applies from combo 2 to combo 3, as when user enters correct input within 5 seconds of correct answer, the user gets 300 points added on.

The maximum additional points one can gain from combos is 300 points, so a user can keep being at combo 3 by continuously entering correct inputs 5 seconds apart from each other.

Once the combo is in play, the combo countdown is set for 5 seconds, so the user has visibility on what is occuring.

Hint:

The hint appears after 30 seconds of idle game time. The user has the option of choosing it at the cost of deducing points, as it reveals how many incorrect words are remaining in the text.

Game End

In order to end playing the game the user can press the stop button. Once the stop button is clicked, the user will be presented a modal giving an overview on how they did and also a functionality to share their results.
